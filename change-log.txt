- Load new browser window for each test
- This CHANGELOG file to hopefully serve as an evolving example of a standardized open source project CHANGELOG.

- Typescript supportive framework design
- Typescript log4j support

- Fully supported protractor api
- Test design using jasmine api
- Support basic page object design
- Support data driven test cases
- Integration with third party dashboard
- Test suite level execution support
- Introduce custom exceptions handling
- Protractor wrapper helper class for navigation, waits, action and asserions
